# 彩票记录与分析工具 - 手机App版

基于Python Kivy框架开发的跨平台手机应用，支持双色球和大乐透的购彩记录、自动对奖、中奖统计和智能随机选号。

## 功能特性

### 1. 购彩记录管理
- 记录每次购彩的日期、彩票类型、号码、投注金额、购彩渠道
- 查看历史记录列表
- 支持手动输入或导入号码

### 2. 开奖查询与自动对奖
- 自动联网获取最新开奖号码（支持多个数据源）
- 按日期查询历史开奖结果
- 自动比对未对奖记录，计算中奖金额
- 内置双色球/大乐透奖金规则表

### 3. 中奖统计与分析
- 累计投入金额统计
- 累计中奖金额统计
- 净盈亏计算
- 中奖率分析
- 各奖级中奖次数统计
- 可视化统计报表

### 4. 智能随机选号（24人算法）
- 基于特殊算法的随机选号功能
- 模拟24人各写一个数字，计算平均值作为基础号
- 若超出上限则取余数，保证号码在有效范围内
- 每个号码位置独立计算，确保随机性

## 支持的彩票类型

### 双色球
- 红球：6个，范围 1-33
- 蓝球：1个，范围 1-16

### 大乐透
- 前区：5个，范围 1-35
- 后区：2个，范围 1-12

## 技术架构

- **前端框架**: Kivy 2.3.0 (Python跨平台UI框架)
- **数据存储**: SQLite本地数据库
- **网络模块**: requests + 多数据源备用
- **核心算法**: Python实现24人随机算法
- **运行环境**: Python 3.7+，支持Windows、Android、iOS

## 安装与运行

### 环境要求
- Python 3.7 或更高版本
- pip包管理工具

### 安装步骤

1. 克隆或下载项目到本地
   ```bash
   cd LotteryMobileApp
   ```

2. 安装依赖包
   ```bash
   pip install -r requirements.txt
   ```

3. 运行应用
   ```bash
   python main.py
   ```

### 打包为手机App (Android)
使用Buildozer工具打包：
```bash
# 安装buildozer
pip install buildozer

# 初始化配置
buildozer init

# 编辑buildozer.spec文件，配置应用信息

# 打包APK
buildozer android debug
```

## 24人算法详解

### 算法步骤
1. 模拟24个人，每人写一个数字（范围：-500到500）
2. 将24个数字相加得到总和
3. 总和除以24得到平均值，取整作为基础号码
4. 如果基础号码超出该位置的上限：
   - 计算基础号码除以上限的余数
   - 如果余数为0，则取上限值
   - 否则取余数作为最终号码
5. 如果基础号码在范围内，直接使用

### 示例
假设红球上限为33，24人数字之和为100：
- 平均值 = 100 ÷ 24 ≈ 4.17 → 取整为4
- 4在1-33范围内 → 最终号码为4

假设蓝球上限为16，24人数字之和为500：
- 平均值 = 500 ÷ 24 ≈ 20.83 → 取整为21
- 21超出1-16范围 → 余数 = 21 % 16 = 5
- 最终号码为5
